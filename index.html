排版配置

InkScript 1.9.5

基本结构

第一个块为全局配置，之后每个 === 分隔一页。# 或 // 开头为注释。

页面类型

类型说明
cover封面页，居中显示标题和副标题
poem诗歌页（默认），显示标题、作者、正文
collection合集页，一页内排多首诗

字段一览

字段用法
标题 / Title页面标题
副标题封面副标题
作者 / Author作者名
日期 / Date创作日期
正文 / Content诗歌正文，支持多行；设为 none 可留空
序言 / Pre正文前的引言
脚注 / Foot页面底部注释
id / ID引用诗歌数据库中的诗歌 ID（需部署 poems.json）
主题 / Theme水墨配色：classic cyan ochre peach bamboo aurora
Seed / 种子固定水墨随机图案，相同值 = 相同背景

显示控制

指令效果
紧凑 / tight散文排版模式，合并行、首行缩进
!显示标题隐藏标题（同理：!显示作者 !显示日期 !显示脚注）
显示子标题在合集页中显示每首诗的标题

合集列表语法

列表: ID1, ID2[脚注:xxx], ID3[紧凑|!显示标题]
用逗号分隔 ID，方括号内覆盖属性，| 分隔多项。

快捷键

Ctrl+Enter / Cmd+Enter 渲染预览

分割: === 列表: ID[k:v] 注释: #...
